## 6. Function Analysis

Given the quadratic function:
$$f(x) = 3x^2 - 12x + 7$$

We need to identify any local maxima or minima.

---

### Solution Steps

1. **Find the derivative $f'(x)$:**
   $$f'(x) = \frac{d}{dx}(3x^2 - 12x + 7) = 6x - 12$$

2. **Find the critical point:**
   Set the derivative equal to zero to find the stationary point:
   $$6x - 12 = 0$$
   $$6x = 12 \implies x = 2$$

3. **Verify the nature of the point:**
   Use the second derivative test:
   $$f''(x) = \frac{d}{dx}(6x - 12) = 6$$
   Since $f''(x) = 6 > 0$, the function is concave up at $x = 2$, which indicates a **local minimum**.

4. **Calculate the function value at $x = 2$:**
   $$f(2) = 3(2)^2 - 12(2) + 7$$
   $$f(2) = 3(4) - 24 + 7 = 12 - 24 + 7 = -5$$

> **Conclusion:** The function has a **local minimum** at $(2, -5)$.
